O que estou fazendo agora e metas para o futuro

Neste exato momento, estou concentrado em buscar meu primeiro emprego como programador para entrar no mercado. Além disso, estou planejando retornar ao curso de Sistemas de Informação para concluí-lo.

Atualmente, estou desenvolvendo uma aplicação web de pomodoro usando React no front-end e Node.js com Express e MongoDB (Mongoose) no banco de dados. Os usuários podem criar um perfil e estabelecer objetivos que serão usados no sistema de pomodoro, registrando o tempo dedicado a cada objetivo.

Além do Pomodoro, estou iniciando o desenvolvimento de uma API RESTful para um projeto de e-commerce no meu portfólio. Também estou trabalhando em uma continuação do meu jogo SimonSays, aplicando o conhecimento adquirido para aprimorar o que poderia ter sido feito de maneira mais eficiente.
